Connect4

Player Turn

Take turns and be the first to Connect4!

New Game

Instructions: Take turns with a friend using the mouse to drop disks into the rack. Connect 4 in any direction to win!

This game has been created using the p5js programming library. Inspiration for this project comes from Daniel Shiffman @shiffman in his amazing tutorial videos. Concept based on the well known original.
